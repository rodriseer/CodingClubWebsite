Python Programming Basics

Learn the foundations of Python programming, including:

  • Variables and data types

      Firstly, what is a 'variable' in the Python (or any other programming language) world?

      In Python, a variable is like a container that holds data.

      Think of it as a label for a value. Instead of remembering the value itself, you give it a name (the variable) and use that name to refer to the value later.

      You create a variable by simply giving it a name and assigning it a value with the "=" sign:

      name = "Your Name" A variable storing the string of your name
      age = 22 A variable storing the integer 22, but you can use any integer here, even if you are a 300 years old vampire

      Data Types:
      - Integers (int): Whole numbers, positive or negative.
      - Floats (float): Numbers with decimals.
      - Strings (str): Text enclosed in quotes.
      - Booleans (bool): Values like 'True' or 'False'.
      - Lists (list): Collections of items in a specific order.
      - Dictionaries (dict): Key-value pairs for storing data.

      In Python, you don't need to declare the variables data type like in other programming languages.
      But keep in mind that each variable data type interacts differently with the code.

  • Control structures (if, for loops, etc.)

      Control structures allow you to control the flow of your program based on conditions.

      If Statements:
      Used to run code only if a certain condition is true. For example:
      if age > 18: print("You are an adult!")

      What is happening here is that the code is comparing if its REALLY TRUE that you are OVER 18
      The "if" comparissons works by comparing two variables (usually input vs stored variable) and then making a decision if its TRUE or FALSE
      When its declared to FALSE, it is reccomended to handle this through an ELSE, for example, if you are under 18 the code would declare FALSE and the ELSE statement would handle this:
      else: print("You are under 18!")

      Loops:
      Loops are used to repeat a block of code. For example, a 'for' loop allows you to iterate over a list of items: for color in favorite_colors: print(color)

  • Basic functions and error handling

      Functions are reusable blocks of code that make your programs modular and more manageable.

      Functions:
      A function allows you to group a set of instructions under a name, making it easier to reuse and organize code. Here’s an example of a simple function:
      def greet(name):
      print("Hello, " + name + "!")
      You can then call this function as many times as you need: greet("Your Name") This will output: Hello, (whatever your name is)!

      Error Handling:
      In Python, you can handle errors gracefully (I really liked this word when I first learned it) using 'try' and 'except' blocks:
      try:
         print(10 / 0)
      except ZeroDivisionError:
         print("Cannot divide by zero!")

Computer and Programming Reading Recommendations

Here are some recommended books for learning programming:

  • Python Crash Course by Eric Matthes

    If you're looking for a fast-paced book that covers all the basics of Python and guides you through hands-on projects, this is the perfect book for you.

    Why read this book?

    • Covers fundamental Python concepts efficiently.
    • Includes engaging projects, such as game development and web applications.
    • Features a major web-based project with deployment using Django.

    What's special about Django?
    Django is a powerful web framework widely used by companies for backend development in web applications. It's a fantastic tool to learn if you're aiming to build secure web applications.



  • Automate the Boring Stuff with Python by Al Sweigart

    If you're looking for a Python book that covers all the basics of Python and guides you through useful and practical projects, this is the perfect book for you.

    Why read this book?

    • Covers fundamental Python concepts efficiently.
    • Includes engaging projects, such as automating repetitive tasks like updating spreadsheet cells, sending emails, and more.
    • Focuses on practical, real-world projects that can save you time.



  • Object-Oriented Python by Irv Kalb

    If you're looking for a Python book that covers the Object-Oriented Programming paradigm, this is a great start.

    Why read this book?

    • Teaches the fundamental Object-Oriented Programming concepts.
    • Includes many OOP projects.
    • Features a major project that will teach you how to implement OOP concepts and user interactions in a Python application.

    What is Object-Oriented Programming?
    Object-Oriented Programming (OOP) is a programming paradigm that organizes code into objects, which represent real-world entities. Each object can contain data (attributes) and methods (functions) that define its behavior. This approach makes it easier to model complex systems by breaking them down into smaller, manageable components.

Hackathons

What is a 'Hackathon'?

What is a Hackathon:

  • A hackathon is a collaborative event where people come together to rapidly develop software or hardware solutions. It's often focused on a specific problem, theme, or technology. Think of it as a marathon for hackers, where teams work intensively over a short period (usually a few days) to create a functional prototype or proof-of-concept.

What a Hackathon is not

What a Hackathon is not:

  • A hackathon is not a competition to see who codes better, faster, or fancier.
    The focus of hackathons isn't solely on writing code, but on teamwork, creativity, and solving real-world problems by developing a working prototype within a short timeframe.
    It’s not a formal event but rather a high-energy space for rapid innovation and experimentation.
